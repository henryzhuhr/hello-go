## 数据结构

### 数组 array

数组是定长的数据结构，长度指定后不能改变。数组的元素类型必须一致

如果事先就知道了要存放数据的长度，且后续使用中不会有扩容的需求，就可以考虑使用数组，

> [!TIP]
> Go 的数组是**值类型，而非引用**，并不是指向头部元素的指针。因此将数组作为参数传递给函数时，由于 Go 函数是传值传递，所以**会将整个数组拷贝**。

#### 数组初始化

```go
array1 := [3]int{}             // 定义数组
array2 := [3]int{1, 2, 3}      // 初始化数组
array3 := [...]int{1, 2, 3}    // 初始化数组，不指定长度
array4 := [5]int{1: 10, 3: 30} // 指定下标初始化

// 遍历数组
for i := 0; i < len(array4); i++ {
    fmt.Print(array4[i], ", ")
}
for j, v := range array4 {
    fmt.Print(j, ":", v, ", ")
}
```

#### 切割数组

切割数组的格式为 `arr[a: b]`，切割的区间为**左闭右开**（`[a,b)`），例子如下

```go
nums := [5]int{1, 2, 3, 4, 5}
nums[1:]  // 下标范围 [1,5) -> [2 3 4 5]
nums[:5]  // 下标范围 [0,5) -> [1 2 3 4 5]
nums[2:3] // 下标范围 [2,3) -> [3]
nums[1:3] // 下标范围 [1,3) -> [2 3]
```

数组在切割后，就会变为切片类型，切片是一个引用类型，指向数组的某个片段，切片的长度和容量可以动态变化。
```go
nums := [5]int{1, 2, 3, 4, 5}
fmt.Println("Type: %T\n", nums)      // Type: [5]int (数组类型)
fmt.Println("Type: %T\n", nums[2:4]) // Type: []int  (切片类型)
```
